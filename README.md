# A simple way to display the avatar of a discord user from an id in your terminal

## Ways to use
### Interactive
```
terminal-dc-pfp-viewer
```

### Non interactive
```
terminal-dc-pfp-viewer --id 556610358396321840
```

## Notes
### The tool uses the API of a website called https://www.discordpfp.gg
### I'm not associated with https://www.discordpfp.gg, the tool can break easily
### if they decide to update their api
